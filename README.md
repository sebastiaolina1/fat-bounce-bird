# fat bouncy bird
Fat Bouncy Bird is a game created by me and only me, inspired by the famous Flappy Bird; however, this time the bird is overweight, which allows it to bounce on the ground and pass through fragile building structures. Because it is heavy, the game makes the bird feel "slow."

## controls
The controls are very simple and limited; use "space" to jump, and the "right mouse button" or "left mouse button" for the same. Remember that the bird can bounce on the ground.

## how to use/install
Go to the demo link and download the zip file. After downloading, locate it in your downloads folder and extract everything. Do not delete anything already in the folder or remove the game from there. It's essential that the files are in the same folder. Locate the file with the Unity logo, probably labeled "Fat Bouncy Bird," click on it, and enjoy the game.
## more about
It's a game made in Unity. If you want to modify it, feel free. Just go to the game's repository link and use it as a Unity project.
